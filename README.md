# node_parser_html
Node.js: Parse html - client server app


###INSTALL

1. Get files

   ```shell
   $git clone https://github.com/cmsrs/node_parser_html.git
   $cd ./node_parser_html
   ```

2. Installing dependents 

   ```shell
   $npm install
   ```

3. Start server

   ```shell
   $npm start
   ```

4. Test app 

   ```shell
   $npm test
   ```

5. bash test

   ```shell
   $cd scripts
   $./post.sh
   $./get
   ```
